Trabajo final sobre cambio climatico


#Autor
Carlos Eduardo Gonzalez Treviño

Artículos y Reportes Automáticos
El sistema genera mini-artículos de investigación con análisis de tendencias climáticas.

Estos reportes pueden ser usados por ONGs, escuelas o medios.

Ejemplo: “En Monterrey, el uso de transporte público podría reducir 15% las emisiones urbanas”.
